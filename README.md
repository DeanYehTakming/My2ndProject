# My2ndProject

## 猜數字遊戲 (Number Guessing Game)

這是一個簡單有趣的猜數字遊戲，用 Python 編寫。

### 遊戲規則

1. 程式會隨機產生一個 1 到 100 之間的整數
2. 玩家輸入猜測的數字
3. 程式會提示「大了」或「小了」來幫助你調整猜測
4. 猜對時會顯示「恭喜通過！」並結束遊戲
5. 遊戲會記錄並顯示你的猜測次數，幫助你看到進步

### 如何執行遊戲

```bash
python3 guess_number.py
```

或者

```bash
python guess_number.py
```

### 遊戲截圖範例

```
==================================================
歡迎來到猜數字遊戲！
我已經想好一個 1 到 100 之間的數字
請開始猜測吧！
==================================================

請輸入你猜測的數字 (1-100): 50
大了！這是你的第 1 次猜測。

請輸入你猜測的數字 (1-100): 25
小了！這是你的第 2 次猜測。

請輸入你猜測的數字 (1-100): 37

==================================================
🎉 恭喜通過！🎉
答案就是 37！
你總共猜了 3 次。
==================================================
```

### 功能特點

- ✅ 隨機產生 1-100 的數字
- ✅ 提供「大了」、「小了」的提示
- ✅ 追蹤猜測次數
- ✅ 輸入驗證（只接受 1-100 的數字）
- ✅ 錯誤處理（非數字輸入、Ctrl+C 中斷）
- ✅ 友善的中文界面

### 技術細節

- 使用 Python 3
- 使用 `random` 模組產生隨機數
- 包含完整的輸入驗證和錯誤處理
